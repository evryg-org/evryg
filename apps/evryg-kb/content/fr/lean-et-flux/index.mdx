# Lean et Flux

Le Lean software delivery applique les principes du Toyota Production System au développement logiciel. Cette section explore comment optimiser le flux de valeur.

## Thèmes abordés

- **Shift Left** : détecter les erreurs au plus tôt
- **Déploiement continu** : f(git) = production
- **Organisation** : Kanban, pull requests, architecture et déploiement
