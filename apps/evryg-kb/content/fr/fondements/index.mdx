# Fondements

Les structures mathématiques et les concepts fondamentaux qui sous-tendent la programmation moderne. Comprendre ces abstractions permet de reconnaître des patterns récurrents.

## Thèmes abordés

- **Structures** : monoïdes, DAGs
- **Dualités** : code et data, description et interprétation
- **Abstraction** : DSLs, injection de dépendances
